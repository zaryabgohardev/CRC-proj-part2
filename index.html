Student Primary User
Responsibilities
  • Browse default feed of upcoming events
  • Filter by tags, date, location, major/school
  • Sort feed by date, popularity, recently added
  • Search for upcoming and past events by keyword
  • Follow topics & organizers
  • RSVP, cancel RSVP, join waitlist

  • Export events to personal calendar (.ics)

  • Set interest preferences on onboarding & update in profile
